# Thinkpad-L490-hackintosh

Thinkpad-L490 OpenCore EFI 适用MacOS Ventura 13.7.8

使用OpCore-Simplify构建，网卡选择airportitlwm，实测wifi与蓝牙均可以正常使用。

OpenCore 1.0.5

硬件配置：

	1. CPU: i5-8250U
	2. 内存：8Gx2
	3. 硬盘：512G ssd
	4. 显卡：核心显卡Intel UHD Graphics 620
	5. 网卡：Intel 9260
